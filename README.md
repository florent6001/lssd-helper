# LSSD-Helper

LSSD-Helper is a tool that help LSSD on GTA.World server, generating paperwork with forms.
## Demo

http://lssd-helper.vercel.app/
## Installation

Install the project with yarn

```bash
  git clone https://github.com/florent6001/lssdhelper
  cd lssdhelper
  yarn
  yarn start
```
